  # Gestión de Usuarios - Evaluación Módulo 5

Este proyecto es una aplicación web interactiva que consume datos de una API externa para gestionar y visualizar información de usuarios. Ha sido desarrollado como parte de la evaluación final del Módulo 5.

## 🚀 Funcionalidades

La aplicación permite realizar diversas operaciones con los datos obtenidos de la API [JSONPlaceholder](https://jsonplaceholder.typicode.com/users):

- **Listado Completo:** Muestra los nombres de todos los usuarios cargados.
- **Búsqueda por Nombre:**
  - Información básica (Email y Ciudad).
  - Información de contacto (Dirección detallada).
  - Información avanzada (Username, Email, Web y Compañía).
- **Gestión de Empresas:** Lista las compañías junto con sus eslóganes (*catchphrase*).
- **Ordenamiento:** Permite visualizar los nombres de los usuarios ordenados alfabéticamente.

## 🛠️ Tecnologías utilizadas

- **HTML5:** Estructura semántica del panel de control.
- **CSS3:** Diseño responsivo y estilizado con gradientes y sombras.
- **JavaScript (ES6+):** - Programación Orientada a Objetos (Clases).
  - Consumo de API con `fetch`.
  - Manejo de asincronía con `async/await`.
  - Manipulación dinámica del DOM.
